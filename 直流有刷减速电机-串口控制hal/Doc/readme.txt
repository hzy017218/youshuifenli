【*】 程序简介 
-工程名称：直流有刷减速电机-串口控制
-实验平台: 野火STM32 F103 拂晓开发板


【 ！】功能简介：
通过串口发送指令,设置电机方向和速度.
指令示例:
d 0
v 1000
方向正转,速度设置为1000,注意指令结尾需要换行,或在串口助手中勾选发送新行

【 ！】实验操作：
/*******************************************************************************
	******************************注意***************************
*******有刷的例程可以接用在有刷驱动板上或者是L298N驱动板板上************
************************************************************************************/
有刷驱动板与电机板的接线
接线:
电机驱动板 5V\GND 		<----> 开发板 5V\GND
电机驱动板 PWM1\PWM2 	<----> 开发板 PE9\PE11
电机驱动板 SD 		<----> 开发板 PD15

L298N与电机板的接线以及电机的接接线
接线:
	L298N驱动板的GND-----------------开发板GND
	L988N驱动板的PWM1---------------开发板PE9
	L298N驱动板的PWM2--------------开发板的PE11
	
电机的M+脚<--------------------->L298N的电机A输出1脚
电机的M-脚<---------------------->L298N电机A输出的2脚


下载本程序，复位开发板即可。

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

定时器TIM1输出PWM信号:
两路PWM输出到PE9\PE11上连接到驱动板上的PWM信号输入.
电机驱动板 PWM1\PWM2 <----> 开发板 PE9\PE11

电机使能引脚:
电机驱动板 SD <----> 开发板 PD15
